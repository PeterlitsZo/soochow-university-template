Soochow University Teamplate
===============================================================================

苏州大学 ACM/ICPC 模板。

提交格式
-------------------------------------------------------------------------------
header 保证 `<type>: <title>`。其中 `type` 包括了：

* `doc`：README 等帮助文档。
* `feat`：增加新东西。
* `style`：修改代码格式。
* `fix`：修改 bug。
* `chor`：辅助工具的改动（如 gitignore）。
* `rfcr`：重构。

保证提交最小化。即一个提交只能干一件事情。

单单 header 概括不了的话，那么一定要写 body 部分。现在的负责是对将来的保证！
